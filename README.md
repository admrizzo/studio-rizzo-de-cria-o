# Studios de Criação

Template white-label da **ia.naimobiliária**, produto **Studios de Criação**.

Ambiente onde a imobiliária importa imóveis do seu CRM e gera vídeos curtos, artes editoriais e legendas prontas para Instagram, WhatsApp e Meta Ads.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Lovable Cloud (backend, auth, storage, edge functions)

## Modo demonstração

Acesse `/demo` para experimentar sem login, com 10 imóveis fictícios pré-carregados.

## Configuração

Após criar uma conta, configure em **Configurar Marca**:

- Nome da imobiliária, telefone, WhatsApp, logo, cores
- URL do feed do seu CRM (RobustCRM ou compatível)
